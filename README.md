# Sify Langfuse SDK

A lightweight, standalone Python SDK for **manual, model-agnostic observability** using **Langfuse**.

This SDK allows developers to trace:
- LLM calls (any model, any provider)
- API calls
- Application logic



---
-  Model-agnostic (OpenAI, Groq, Hugging Face, local LLMs, APIs)
-  User-level and session-level tracking
-  Native Langfuse Observations API
-  No monkey-patching
-  Works with Langfuse Cloud & Local
